# Tichi QA Internship Assignment

This repository contains my QA internship assignment for the Tichi web application.

The project covers manual testing, defect reporting, and Playwright automation for the Login functionality.

## Application

https://tichi-app-webapp-stage.web.app

## What's Included

- Login Test Cases
- Signup Test Cases
- Defect Reports
- Playwright Automation Scripts
- Test Screenshots

## Technologies Used

- Python
- Playwright
- VS Code
- Google Chrome

## Automated Test Scenarios

- Valid Login
- Invalid Email
- Invalid Password
- Empty Email
- Empty Password

## Manual Test Coverage

### Login
- Valid Login
- Invalid Login
- Email Validation
- Password Validation
- Required Field Validation

### Signup
- Valid Signup
- Required Field Validation
- Email Validation
- Password Validation
- Confirm Password Validation

## How to Run

Install Playwright:

```bash
pip install playwright
playwright install
```

Run a test:

```bash
python login.py
```

or

```bash
python valid_login.py
```
